<h1 align="center">Installation</h2>

<p align="center"><strong>You have two ways to install the mod. You can do it through Nexus Mods:</strong></p>

<p align="center">
  <a href="https://www.nexusmods.com/mywintercar/mods/1478">
    <img alt="Nexus Mods Badge" src="https://img.shields.io/endpoint?url=https%3A%2F%2Fgist.githubusercontent.com%2Fradj307%2Fe9a80731ee236cc67fb00b698e75201e%2Fraw%2F5230074dfb1a60fba917a1232f9382fa5cfec5db%2Fendpoint.json&style=for-the-badge">
  </a>
</p>

<p align="center"><strong>Or you can do it here, through GitHub:</strong></p>

<p align="center">
1. Download the latest version from <a href="https://github.com/xAwakenDeveloper/Snowy-Diagnostics/releases">Release</a>.<br> <br>
2. Install <a href="https://github.com/piotrulos/MSCModLoader">MSCLoader (For My Winter Car)</a>.<br> <br>
3. Extract the files into your mods folder.<br> <br>
4. Launch the game and enjoy the mod!
</p>
